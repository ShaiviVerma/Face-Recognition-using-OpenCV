# Face Recognition using OpenCV

This project performs real-time face recognition using Python and OpenCV with a live webcam.

## Features
- Detects faces in real time
- Recognizes trained faces
- Uses LBPH algorithm

## How to Run
1. Install Python
2. Install libraries:
   pip install -r requirements.txt
3. Run:
   python face_recognition.py
